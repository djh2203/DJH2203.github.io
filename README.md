# DJH2203.github.io

这是我的 GitHub Pages 仓库说明文档。

## 仓库概览

- 根目录 `index.html`
  - 一个个人主页风格的单页网站。
  - 页面内容包含个人介绍、学校/专业、年级、项目 `PhonePhoenix` 和联系方式，使用了 WebGL 海面/天空背景和滚动分场景效果。

## 文件夹说明

- `2203/`
  
  - 一个班级主页，是“青岛九中高二三班”的相关页面。
  - 页面主要包含班级行政/班委信息、学习资料入口，以及班级文化图片展示。

- `articles/`
  
  - 用于放文章或文章相关资源。

- `game/`
  
  - 一个以“青岛九中的拯救大冒险”为主题的网页小游戏。

- `gamemgb/`
  
  - 另一个网页小游戏入口，是 `game/` 的一个魔改版，带有更强的整活风格。

- `images/`
  
  - 图片资源目录。

- `lixian2203/`
  
  - 一个较早的班级主页。“青岛九中高一三班”相关页面。
  - 页面包含班级行政组成、学习资料、优秀成员展示和班级文化视频/链接。

- `projects/`
  
  - 这是一个项目集合目录。

- `projects/ADs/`
  
  - 页面里有会移动的广告 iframe，把“青岛九中高一三班的图片”做成一个动态广告展示页。

- `sr/`
  
  - 一个基于 Scratch 导出的网页小游戏项目。

- `srjump/`
  
  - 也是一个基于 Scratch 导出的网页小游戏项目。

- `starwar/`
  
  - 一个网页射击游戏，修改自[2starwar.top](http://2starwar.top/)(已不可访问)
  - 页面标题和 HUD 表明它是“星球大战/太空射击”风格作品，并且注明是汉化或重制版本。
